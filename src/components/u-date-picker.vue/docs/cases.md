### 日期范围复杂情况

``` html
<u-date-picker
    range
    picker="date"
    width="large"
    minDate="2023-01-01"
    maxDate="2023-07-31"
    start-date="2023-03-10"
    end-date="2023-03-20"
    preIcon="home"
    suffixIcon="time"
    clearable>
</u-date-picker>
```

### 日期范围禁用

``` html
<u-date-picker
    range
    picker="date"
    start-date="2023-03-10"
    end-date="2023-03-20"
    disabled>
</u-date-picker>
```
