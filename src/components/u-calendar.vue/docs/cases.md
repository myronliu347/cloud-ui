### 日期范围最大最小值

``` html
<u-calendar-range
    picker="date"
    start-time="2023-03-10"
    end-time="2023-03-20"
    min-date="2023-03-02"
    max-date="2023-03-25">
</u-calendar-range>
```

### 日期范围全部禁用

``` html
<u-calendar-range
    picker="date"
    start-time="2023-03-10"
    end-time="2023-03-20"
    disabled>
</u-calendar-range>
```
